# 💪 Smart Gym Management System

A full-stack **Gym Management System** built using **Flask (Python)** and **MySQL**.  
This project helps gym administrators manage members, track attendance, handle membership plans, and monitor payments efficiently.

---

## 🚀 Features

- 👤 Member Management (Add, Edit, Delete, View Profile)
- 🔁 Membership Renewal System
- 📅 Attendance Tracking (Daily Check-in)
- 💳 Payment Management with History
- 📊 Dashboard with Analytics (Members, Expiry, Revenue)
- ⚠️ Expiry Alerts for Members
- 📁 Export Members Data (CSV)
- 🎨 Modern Responsive UI (Bootstrap + Custom Design)

---

## 🛠️ Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** MySQL
- **Libraries:** Chart.js, Flask, MySQLdb
